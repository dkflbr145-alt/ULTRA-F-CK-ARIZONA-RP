Этот скрипт расширяет функционал клиента для сервера Arizona RP, делая игру удобнее — без нарушения правил! ✨
Написан на Lua, совместим с MoonLoader и SAMPFUNCS.

Как установить 📦

🔥🔥🔥Поместите файл в папку Arizona Games Launcher\bin\arizona\moonloader. 🔥🔥🔥

Активация:
/ff или зажать R + G

На данный момент в скрипте 190+ функций.

Основные функции 🌟

Автоматизированное управление инвентарём 🎒

Уведомления о доступных заданиях и событиях 🔔

Улучшенный интерфейс для взаимодействия с NPC 👥

Оптимизация производительности (снижение нагрузки на клиент) ⚙️

Требования 🧩

SAMPFUNCS 5.4.1

MoonLoader 0.26

Важно ⚠️

Скрипт не модифицирует игровые файлы.

Даёт преимуществ в PvP или экономике.
